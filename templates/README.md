# Security Risk Assessment Report Templates

Reusable templates for documenting security risk assessments focused on physical security, operational security, wireless security, and infrastructure security reviews.

## Templates Purpose

This templates were designed to provide a consistent structure for:
- qualitative risk assessments,
- infrastructure security reviews,
- operational security analysis,
- physical and wireless security assessments,
- remediation planning.

## Included Sections

- Executive Summary
- Scope and Purpose
- Methodology
- Asset Identification
- Existing Security Controls
- Findings
- Risk Assessment
- Remediation Strategy
- Residual Risk
- Assumptions and Limitations

## Framework References

The templates structure was inspired by:
- NIST SP 800-30 Rev.1
- NIST SP 800-53 Rev.5
- OWASP Risk Rating Methodology

## Notes

These templates are intentionally lightweight and adaptable for small-scale environments, educational projects, and practical security assessments.

It is not intended to replace formal enterprise audit or compliance documentation.
