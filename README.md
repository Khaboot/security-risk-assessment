# Security Assessments Portfolio

Collection of practical security assessments, risk analyses, and infrastructure security reviews focused on small-scale and real-world environments.

## Focus Areas

- Risk assessment
- Wireless security
- Physical security
- Operational security
- Network segmentation
- Access control
- Security documentation

## Repository Structure

Each assessment contains:
- report documentation
- methodology
- supporting evidence
- diagrams
- remediation recommendations

## Disclaimer

Assessments are based on real-world environments. Sensitive details, locations, credentials, and identifiers were redacted for privacy and security reasons.
